#  Prog1 - Java Fundamentals

Bu depo, üniversite 1. sınıf Programlama Dilleri dersi kapsamında Java ile geliştirilen temel seviye projeleri ve algoritma alıştırmalarını içermektedir. Bu kodlar; temel programlama mantığını, kontrol yapılarını ve algoritmik düşünme becerilerini pekiştirmek amacıyla yazılmıştır.

##  İçindekiler (Projeler)

Aşağıdaki listeden ilgili projenin klasörüne giderek kodları ve detayları inceleyebilirsiniz:

* **[ATM Projesi](./ATM-Project--main):** `switch-case` ve `if-else` yapıları kullanılarak geliştirilmiş, temel bankacılık işlemlerini (para çekme, yatırma, bakiye sorgulama) simüle eden konsol tabanlı uygulama.
* **[Not Hesaplama Sistemi](./GradeCalculator):** Vize/Final ağırlıklı ortalama hesaplayan ve final barajı mantığına sahip Java algoritması.
* **[Word Guessing Game](./Word%20Guessing%20Game):** String işlemleri ve döngü mantığı ile geliştirilmiş interaktif kelime tahmin oyunu.
* **[E-Ticaret Envanter Sistemi](./E-Ticaret-Envanter-Sistemi):** API'den veri çeken, dosya işlemleri (File I/O) ve hata loglama yapabilen kapsamlı envanter yönetim aracı.

##  Kullanılan Teknolojiler ve Konular
* **Programlama Dili:** Java
  
* **Temel Konular:**
  * Karar Yapıları (If-Else, Switch-Case)
  * Döngüler (For, While, Do-While)
  * Diziler (Arrays) ve Koleksiyonlar (ArrayList)
  * Metotlar (Functions) ve String İşlemleri
    
* **İleri Seviye Konular (Yeni!):**
  * Dosya İşlemleri (File I/O - Okuma ve Yazma)
  * Hata Yönetimi (Exception Handling - Try/Catch)
  * API Entegrasyonu (`HttpURLConnection` ile webden veri çekme)
  * Veri Ayrıştırma (String Parsing)
